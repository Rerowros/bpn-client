# BPN Client Trademark And Attribution Policy

This policy records the BPN Client brand and attribution requirements used together with `LICENSE.md`.

## Official Project And Service Link

When distributing the original or a modified version of BPN Client as a ready-to-use application, the distributor must preserve a clearly available link or button to the official BPN VPN service.

Official VPN service:

```text
https://t.me/bpn228bot?start=c_desktopApp
```

The link or button must be available to users without reading the source code. Acceptable locations include the About screen, Settings, Help menu, main window footer, or another similarly visible user-interface location.

Forks and modified versions may add their own links and services, but they must not remove, hide, disguise, or replace the official BPN service link without preserving a clearly available link to the official BPN VPN service.

## Forks And Modified Versions

Modified versions must clearly state that they are forks or derivative versions and are not the official BPN Client unless the project owner gives separate written permission.

Forks must not imply official endorsement, ownership, or compatibility with the official BPN VPN service unless separately approved.

## Protected Brand Elements

The source code license does not grant permission to misuse:

- the `BPN Client` project name;
- project logos or icons;
- official screenshots and marketing materials;
- official domains and VPN service links;
- names that are confusingly similar to the official BPN Client project or service.

## Attribution

Distributions should preserve the project attribution notice and link to the official project repository when practical.
